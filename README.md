# Emotion-Based Song Recommender

## Overview
This project implements an Emotion-Based Song Recommender system using computer vision and machine learning. It captures facial and hand landmarks to detect emotions and recommends songs based on the user's emotional state.

## Features
- **Real-time Emotion Detection:** The system captures facial and hand landmarks to detect the user's emotions in real-time.
- **Song Recommendation:** Recommends songs based on the detected emotional state.
- **Interactive Web Interface:** Allows users to interact with the system through a web interface.

## Prerequisites
- Python
- OpenCV
- Mediapipe
- Streamlit
- TensorFlow (for emotion prediction)
- Web browser

## Setup
1. Install the required dependencies using the following command:
   ```bash
   pip install opencv-python mediapipe streamlit tensorflow
2. Run the application:
    ```bash
    streamlit run your_script_name.py

## Usage
- **Launch the application.
- **Allow access to your camera.
- **The system will detect your emotions in real-time.
- **Click the "Recommend Me Song" button to get song recommendations based on your emotions.
